This folder contains files to build getOPERATOR ( mex_getOPERATORS.m )
and  to run an example (getOPERATOR_example)

In order to build and use getOPERATORS() you need to change path to c++ dynamic libraries
and header files to the appropriate ones.

An old MATLAB version of getOPERATORS().m was renamed to get_OPERATORS_old().m to avoid clashes.
Remove mex-ed version of getOPERATORS() and rename the matlab version back to use to use matlab
implementation.

You may want to run getOPERATORS_example() to check the correctness of mex version before using
function elsewhere.
